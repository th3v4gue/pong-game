# Pong Game 🎮

A simple, beginner-friendly Pong game made with Python and Pygame!

## How to Play

### Requirements
- Python 3.x
- Pygame

### Installation

1. Install Python from [python.org](https://www.python.org/)
2. Install Pygame:
```bash
pip install pygame
```

### Running the Game

```bash
python pong.py
```

## Controls

**Left Paddle (Player 1):**
- `W` - Move Up
- `S` - Move Down

**Right Paddle (Player 2):**
- `UP ARROW` - Move Up
- `DOWN ARROW` - Move Down

## Game Rules

- Each player controls a paddle on their side
- The ball bounces between paddles
- If the ball passes your paddle, the opponent scores 1 point
- First to reach a certain score wins!
- Close the window to exit

## Game Features

✅ Two-player gameplay
✅ Score tracking
✅ Ball physics and collision detection
✅ Simple and clean interface
✅ Easy to understand code (great for learning!)

## Code Explanation

The game uses three main classes:

1. **Paddle** - Handles paddle position and movement
2. **Ball** - Handles ball position, movement, and collision
3. **Main Loop** - Controls the game flow, rendering, and input

Perfect for learning game development basics!
